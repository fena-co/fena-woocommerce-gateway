== Description ==

Accept instant bank transfers through Fena on your WooCommerce store.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Fena ECommerce'
3. Activate Fena ECommerce from your Plugins page.
